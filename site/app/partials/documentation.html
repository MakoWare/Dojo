Overview

All Objects (Dispatches, Vehicles, Patients, etc) are Created, Updated, and Deleted the same way.



Figure 1 shows the common interface for creating any of the Objects (currently a Dispatch Object). The green "Add Dispatch" button will create a new Dispatch Object, which will then display the Update interface where changes to the new Dispatch can be made. This workflow is the same for all of the Objects, not just Dispatch.



After the "Add Dispatch" button has been clicked from the previous figure, the system goes to the Update interface. This interface is where all updates to the Objects can be made. Once the blue "Update" button has been clicked the changes to the Object are saved, and the system returns to the Object list view. The red "Delete" button will immediately delete the current Object being edited, then the system will return to the Object list view

Searching and Filtering

All Objects can be searched and filtered in the same manner. Each Object has specific attributes pertaining to the type of Object it is, for example a Dispatch has a pickUp and dropOff time. All Objects can be searched for, or filtered by any of these attributes
